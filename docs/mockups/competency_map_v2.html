PathFinder AI · 역량 분석 #2 가이드

🕸
역량 레이더 차트

파란 채움 영역이 현재 역량, 빨간 점선이 직무 요구 수준입니다. 두 영역의 차이가 클수록 준비가 더 필요한 영역입니다. 축 레이블이나 점을 클릭하면 해당 역량 카드가 강조됩니다.

📊
역량 상세 카드

각 카드를 클릭하면 수치 근거와 면접 액션이 펼쳐집니다. 왼쪽 색 테두리가 역량 상태를 나타냅니다.

  • 🟢 초록 — 어필 가능: 경험이 있고 면접에서 강조할 수 있는 역량
  • 🟡 주황 — 답변 정리: 경험은 있으나 설명 보완이 필요한 역량
  • 🔴 빨강 — 학습 필요: 현재 경험 근거가 부족한 역량
⭐
수치 점수 안내

내 역량 점수는 자소서·이력서 경험 근거를 기반으로 AI가 추정한 값이며, 직무 요구 점수는 채용공고 분석 결과입니다. 정밀 수치가 아닌 상대적 참고 지표입니다.

💪
어필 가능 (Strength)

이미 경험이 있고 면접에서 강조할 수 있는 역량입니다. 구체적인 수치, 프로젝트 이름, 결과를 함께 말하면 효과적입니다.

💡 STAR 구조로 2분 내 답변 연습 권장
📝
답변 정리 (Articulate)

경험은 있지만 설명이 부족할 수 있는 역량입니다. 개념 정리 → 내 경험 연결 → 구체적 사례 순서로 답변 구조를 미리 잡아두세요.

💡 면접 전날 개념 요약 노트 작성 권장
📚
학습 필요 (Study)

현재 경험 근거가 부족한 영역입니다. 솔직하게 인정하되, 학습 계획과 관련 개념 이해를 보여주는 것이 좋습니다.

💡 Python 경험과의 유사 개념으로 연결해 설명
❓
질문 & 답변 전략 토글

각 역량 항목 아래 「질문 & 답변 전략 보기」 버튼을 클릭하면, 일산로보틱스 기술면접에서 나올 수 있는 예상 질문과 답변 전략, 꼬리질문을 확인할 수 있습니다.
