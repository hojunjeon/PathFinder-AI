# 사용자 입력 내용

## 채용공고 입력 (공고마다 사용자가 입력)

### 회사명
### 직무명

### 담당 업무
### 자격 요건
### 우대사항


## 자기소개서 입력 (공고마다 사용자가 입력)

### 자기소개서 항목 1
### 자기소개서 항목 1에 대한 답변
- 항목과 답변 세트 개수는 공고마다 상이


## 프로필 입력 (계정마다 한 번만 사용자가 입력)

### 기본 정보
- 이름
- 전공
- 학력

### 경력 사항
-  회사명
- 직무 
- 주요 업무 및 성과


### 프로젝트
- 프로젝트 명
- 역할
- 기술 스택
- 프로젝트 설명

### 자격증
- 자격증명


### 수상 내역
- 수상명
- 설명
