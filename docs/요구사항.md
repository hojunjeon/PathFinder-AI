# PathFinder AI
- 취준생들의 기업/직무별 취준 로드맵 추천 서비스

---

## 목적
- 서류 합격 이후 면접 유형에 따라 학습/준비 로드맵을 추천해주는 서비스

---

## 기능

### 1. 프로필 페이지
- 계정을 만들어 프로필에 이력사항을 저장

### 2. 분석 생성 페이지
- 서류 합격한 회사/직무의 채용공고 url을 입력
- 서류 접수 시 제출한 자소서 등을 입력
- 면접 유형 선택

### 3. 분석 단계
- 사용자 정보를 종합해서 현재 수준을 파악
- DB에 저장된(미리 딥리서치를 통해 DB나 RAG 를 구성) 기업의 인재상, 면접 방식 등의 정보와 채용공고를 종합해 지원자에게 요구하는 역량을 추출

### 4. 분석 및 로드맵 추천 결과물 페이지
- 사용자 정보, 채용공고 및 기업, 면접 유형, 미리 조사된 DB(or RAG)을 종합해 개인 맞춤형 로드맵 생성.

---

## 제약 사항

### 1. 프론트 백엔드 api
- vue / django / fast api 사용

### 2. GMS_KEY로 llm api를 호출
- 텍스트
```
curl "https://gms.ssafy.io/gmsapi/api.openai.com/v1/chat/completions" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $GMS_KEY" \
  -d '{
    "model": "gpt-5-nano",
    "messages": [
      {
        "role": "developer",
        "content": "Answer in Korean"
      },
      {
        "role": "user",
        "content": "Summarize the 2024 total production data."
      }
    ]
  }'
```

## MVP 이후 추가기능 

### 기업, 직무의 정보를 반영한 DB or RAG 최신화
- 옛 정보로 로드맵 작성하면 의미가 없기 때문

### 이미지 생성 llm 활용
- rpg 요소를 가미한 프로필 이미지 등 생성 하면 재미 있을 듯

### 5. 채용시장 경쟁률 분석 대시보드 (`/dashboard`)
- **기획 배경**: 구직자가 실시간 채용 트렌드(산업별 평균 급여, 연봉 분포, 직급 및 경력 선호도)를 다각도 차트로 분석하여 전략적 지원을 수립할 수 있는 환경 제공.
- **시각화 명세**:
  - **산업별 평균 연봉 vs 평균 지원자 수 (이중 축 혼합 차트)**: 콘텐츠/미디어, 금융 등 각 산업별 연봉수준과 경쟁 인원의 격차 도출.
  - **직급별 지원자 분포 (원형 도넛 차트)**: 신입, 대리, 과장 등 채용 수요가 높은 연차 비중 표시.
  - **경력 요구조건 트렌드 (라인 차트)**: 연도별 혹은 분기별로 채용 공고 내 경력 조건 변화 추이 파악.
  - **급여 분포도 (누적 버블/영역 차트)**: 전체 업계 대비 급여 밴드 분포도 표현.
- **주요 부가 기능**:
  - **인터랙티브 필터**: 산업군 선택, 경력 범위(Slider) 조절, 기업 검색창 조합에 따른 모든 차트의 동시 동적 갱신.
  - **PNG 저장 기능**: 분석된 차트 이미지를 시각화 텍스트 손실 없이 PNG 확장자로 다운로드하여 공유/보관 지원.
