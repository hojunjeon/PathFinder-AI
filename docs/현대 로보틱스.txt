# 자기소개

## 직무(기업) 지원 동기 및 입사 후 계획

유일로보틱스에 지원한 이유는 로봇을 단일 제품이 아니라 제조 현장의 자동화 흐름 안에서 구현하는 회사라고 판단했기 때문입니다. 유일로보틱스는 직교로봇, 다관절로봇, 협동로봇뿐 아니라 생산라인 자동화 솔루션까지 다루며, 다양한 제조 공정에 로봇을 연동하는 방향으로 사업을 확장하고 있습니다. 저는 로봇 제어 SW가 실제 현장에서 안정적으로 작동하는 자동화 시스템의 핵심이라는 점에 관심을 가져왔고, 로봇과 스마트팩토리 솔루션을 함께 개발하는 유일로보틱스에서 그 역량을 발휘하고 싶습니다.

저는 로봇 제어 SW 개발 직무가 단순히 로봇을 움직이는 코드를 작성하는 일이 아니라, 로봇의 동작 구조와 센서 입력, 통신, 상태 확인, 안전 조건을 함께 고려해 신뢰성 있는 제어 흐름을 만드는 역할이라고 이해했습니다. 따라서 이 직무에는 하드웨어 이해를 바탕으로 제어 로직을 구현하는 능력, 실제 동작 결과를 확인하며 개선하는 능력, 시스템 전체 흐름을 보고 문제를 해결하는 능력이 중요하다고 생각합니다.

이와 관련해 로봇 팔 2대를 활용한 프로젝트에서 Python과 ROS 기반으로 Master-Puppet 구조를 구성하고, 관절 제어 및 데이터 수집 환경을 구축했습니다. 이 경험을 통해 로봇 제어는 모터 값 전달만으로 완성되는 것이 아니라, 관절 상태, 카메라 데이터, 실행 환경, 데이터 흐름이 함께 맞물려야 안정적으로 동작한다는 점을 배웠습니다. 또한 Arduino 기반 프로젝트에서는 센서 입력 조건에 따라 모터 동작을 제어하며 실제 하드웨어 반응을 기준으로 제어 조건을 조정하는 경험을 쌓았습니다.

입사 후에는 이러한 경험을 바탕으로 로봇 제어기와 모션 제어, 센서 연동, 테스트 과정에서 실제 장치의 동작을 정확히 이해하고 안정적인 제어 SW를 구현하겠습니다. 특히 개발한 기능이 현장 자동화 흐름 안에서 일관되게 작동하도록 동작 결과를 확인하고 개선하며, 유일로보틱스 로봇 시스템의 신뢰성과 적용성을 높이는 개발자가 되겠습니다.



## 스스로 높은 수준의 목표를 설정하고 열정적으로 달성한 경험

저는 로봇 팔 프로젝트에서 단순히 로봇을 동작시키는 수준을 넘어, 제어 구조와 데이터 흐름을 함께 이해해 안정적으로 동작하는 로봇 시스템을 구현하는 것을 목표로 삼았습니다. 이를 위해 HW 제어 역할에 머무르지 않고, 모방 학습 모델의 동작 원리와 ROS 기반 실행 구조까지 빠르게 이해하며 프로젝트 완성도를 높이고자 했습니다.

해당 프로젝트는 로봇 팔에 모방 학습을 적용해 목표 위치를 인식하고 정밀 동작을 수행하도록 구현하는 과제였습니다. 저는 로봇 팔이 목표 동작을 수행하기 위해 어떤 데이터가 필요한지 이해하고, 이미지, 모터, 시계열 등 실험에 필요한 데이터를 목적에 맞게 수집하는 데 집중했습니다. 또한 ML을 전담한 것은 아니었지만, 모델 코드와 입력 데이터 형식을 분석해 동작 원리를 빠르게 파악했고, 일부 파라미터를 조정하며 모델별 결과를 비교했습니다. 이 과정에서 새로운 기술도 코드 구조와 데이터 흐름을 중심으로 이해하면 필요한 역할을 빠르게 수행할 수 있다는 점을 배웠습니다.

한편, 원격 제어 환경에서 카메라 확인과 로봇 팔 제어를 동시에 수행하기 어려운 문제가 있었습니다. 저는 이를 단순 실행 오류가 아니라 실행 환경과 기능 연결 구조의 문제로 보고 원인을 분석했습니다. SSH 환경의 GUI 제한으로 실시간 확인과 제어가 원활하지 않다는 점을 파악한 뒤, ROS를 적용해 카메라 처리와 로봇팔 제어 기능을 노드별로 분리했습니다. 이를 통해 데이터 수집과 동작 검증을 안정적으로 이어갈 수 있었고, 로봇 제어 SW에서는 개별 기능뿐 아니라 실행 환경과 시스템 구조를 함께 고려해야 한다는 점을 체감했습니다.

저의 강점은 로봇의 동작, 코드 구조, 데이터 흐름을 함께 이해하고 안정적인 제어 흐름으로 연결하는 문제 해결 역량이라고 생각합니다. 유일 로보틱스의 로봇 제어 SW 개발 직무에서도 로봇 제어기, 모션 제어, 센서 연동 과정에서 실제 장치의 동작을 정확히 이해하고, 현장 자동화 환경에서 신뢰성 있게 작동하는 제어 SW 개발에 기여하겠습니다.


## 협업을 통해 공동의 목표를 달성한 경험 (의견차이, 갈등해결, 역할분담 등)

저는 킥보드 프로젝트에서 리더를 맡아 협업이 쉽지 않은 상황에서도 팀원들이 끝까지 함께 움직일 수 있도록 협업 구조를 만들고자 노력했습니다. 각자의 업무가 최종 시제품 완성이라는 공동의 목표로 연결되도록 진행 상황을 공유하고 조율하는 데 집중했습니다.

해당 프로젝트는 2인 이상 탑승과 무분별한 주차 문제를 해결하기 위해 인원 제한 기능과 자율 주행 기능을 갖춘 시제품을 제작하는 과제였습니다. 저는 팀 리더로서 제한된 기간 안에 팀원 모두가 각자의 역할을 수행하고, 최종적으로 실제 동작 가능한 결과물을 완성하는 것을 공동의 목표로 설정했습니다.

당시 팀원들은 인턴과 복수전공을 병행하고 있어 실시간 협업이 쉽지 않았습니다. 저는 프로젝트 흐름이 끊기지 않도록 매주 정기 회의를 운영하며 각자의 진행 상황과 어려움을 공유하도록 했고, 회의 내용을 문서로 정리해 모두가 전체 진행 상황을 파악할 수 있도록 했습니다. 이를 통해 시스템 설계, 회로 구성, 제어 코드 구현이 개별 업무로 분리되지 않고 하나의 시제품 제작 목표 안에서 연결되도록 관리했습니다.

특히 부품 선정은 하드웨어 구조와 예산에 직접적인 영향을 주는 중요한 업무였습니다. 저는 전체 시스템 구조를 기준으로 필요한 부품 사양을 검토하고, 선택지별 장단점과 선정 근거를 문서화했습니다. 이를 통해 팀원들이 같은 기준으로 의사결정할 수 있도록 돕고, 불필요한 재구매를 방지하여 전체 예산의 약 15%를 절감할 수 있었습니다.

결과적으로 프로젝트는 팀원들이 각자의 상황 속에서도 책임감을 가지고 협력한 끝에 성과발표회 수상으로 마무리되었습니다. 이 경험을 통해 협업의 핵심은 역할을 나누는 것에 그치지 않고, 각자의 업무가 공동의 목표로 연결되도록 흐름을 만드는 것임을 배웠습니다. 유일로보틱스의 로봇 제어 SW 개발 직무에서도 제어기, 센서, 모터, 하드웨어 구조 등 여러 요소가 함께 맞물리는 만큼, 전체 흐름을 이해하고 팀과 적극적으로 소통하며 신뢰성 있는 로봇 제어 SW 개발에 기여하겠습니다.



## 직무 강점 및 관련 경험 (교육이수, 프로젝트 경험, 인턴십, 대/내외활동 등)

저는 로봇 제어 SW 개발 직무가 로봇을 움직이는 코드를 작성하는 데 그치지 않고, 하드웨어 동작과 센서 입력, 실행 환경을 함께 고려해 안정적인 제어 흐름을 구현하는 역할이라고 이해하고 있습니다. 이를 위해서는 하드웨어 이해 기반의 제어 SW 구현력이 중요하다고 생각하며, 저는 로봇팔 제어 프로젝트와 자동화 실습을 통해 이 역량을 쌓아왔습니다.

로봇팔 프로젝트에서는 두 대의 로봇팔을 Master-Puppet 구조로 구성해 동기화 코드를 적용했고, 모방 학습에 필요한 이미지, 관절값, 시계열 데이터를 수집하며 로봇 동작과 데이터가 연결되는 흐름을 경험했습니다. 이 과정에서 로봇 제어는 모터 값을 전달하는 것만으로 완성되지 않고, 실제 동작 결과와 데이터, 실행 환경이 함께 맞아야 안정적으로 구현된다는 점을 배웠습니다.

또한 원격 제어 환경에서 카메라 확인과 로봇 팔 제어를 동시에 수행하기 어려운 문제가 있었을 때, ROS의 노드 구조를 활용해 카메라 처리와 로봇 팔 제어 기능을 분리했습니다. 이를 통해 데이터 수집과 동작 검증을 이어갈 수 있었고, 문제를 단순 오류가 아니라 시스템 구조 관점에서 분석하고 개선하는 경험을 쌓았습니다.

WISET 지능형 로봇 과정에서는 PLC 래더 프로그래밍으로 가상의 공정에 대해서 공정 흐름을 제어하는 자동화 제어 설계 프로그램을 수행했습니다. 또한, Python 기반 비전 실습을 통해 이미지 인식, 보정, 거리 측정 기능을 구현했으며, 두산의 협동 로봇의 동작 제어 로직을 설계해 물체 이동 동작을 구현하기도 했습니다. 이를 통해 제어 프로그램, 비전 시스템, 협동 로봇 등의 지능형 로봇 기술의 전반에 대해 이해할 수 있었습니다.

입사 후에는 이러한 경험을 바탕으로 유일로보틱스의 로봇 제어기, 모션 제어, 센서 연동 업무에서 실제 장치의 동작을 정확히 이해하고 구현하겠습니다. 나아가 테스트 결과를 바탕으로 제어 로직을 개선하며, 현장 자동화 환경에서 신뢰성 있게 동작하는 로봇 제어 SW 개발에 기여하겠습니다.
